# Expenso: Expense Tracker 
A full-stack Expense Tracker web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). This project helps users track income and expenses, visualize spending behavior, and manage personal finances efficiently.

---
## Features

User authentication using JWT

Add, update, and delete income transactions

Add, update, and delete expense transactions

Category-based transaction management

Emoji/icon-based category selection

Real-time balance calculation

Interactive charts for income and expenses

Separate dashboard and expense pages

Export data to Excel 

Responsive FinTech-inspired UI

---
## Tech Stack
Frontend : React.js, Tailwind CSS

Backend : Node.js, Express.js, MongoDB

JSON Web Token (JWT)

---
## Installation & Setup
Step 1: Clone Repository
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker

Step 2: Backend Setup
cd backend
npm install
npm run dev

Step 3: Frontend Setup
cd frontend
npm install
npm run dev

---
## Application Flow

1. User registers or logs in securely
2. Dashboard displays balance summary
3. User adds income and expense entries
4. Transactions are stored in MongoDB
5. Charts update dynamically
6. User can delete or manage transactions
7. Logout
